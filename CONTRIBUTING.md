# Contributing to ESN GO

## Current Phase: Demo & Feedback

We're in early stages! Contributions welcome from everyone.

### Ways to Contribute

**Everyone can help:**
- Test the demo and provide feedback
- Report bugs or suggest features
- Share ideas in issues
- Help with content (city guides, translations)

**Developers:**
- Check [open issues](https://github.com/ESNTurkiye/ESN-GO/issues)
- Fork, create a branch, make changes
- Submit a pull request to `demo/v1`

## Workflow

1. **Open an issue** or comment on existing one
2. **Discuss before coding** - save time, align on approach
3. **Create PR** - clear title, reference issue number
4. **Code review** - be patient, be kind

## Questions?

Reach out to the project team or open a discussion.

---

**Remember**: This is volunteer work. Do what you can, when you can. Every contribution matters! ❤️
