## About

ESN GO is a comprehensive digital platform designed to help Erasmus and international exchange students explore Türkiye with confidence. Created by ESN Türkiye, the platform provides curated city guides, practical student survival tips, and personalized discovery tools to make every student's Turkish experience unforgettable.

Whether you're looking for budget-friendly restaurants, planning weekend trips to Cappadocia, or trying to navigate transportation systems, ESN GO connects you with authentic local experiences and peer-reviewed recommendations.

**Current Status**: Demo phase - actively seeking feedback from the community.

## Key Features

### City Guides & Destinations
Curated guides for cities across Türkiye including Ardahan, Bilecik, Bolu, and Ankara. Features intelligent **Vibe Filters** that let you sort destinations by Party, Nature, Culture, or Chill to match your travel style.

### Student Survival Hacks
Real tips from fellow Erasmus students covering essential topics like transportation card setup (Istanbulkart), university cafeteria navigation, and local customs. Learn from students who've been there before.

### Budget Eats
Wallet-friendly restaurant recommendations and comprehensive street food guides. Discover authentic Turkish cuisine without breaking your student budget.

### Event Hub
Stay connected with a centralized calendar for upcoming ESN events, organized trips (including Cappadocia tours), social gatherings, and parties across Turkish cities.

### Additional Features
- Multi-language support for international students
- ESNcard discount integration
- Student reviews and ratings
- Route planning and favorites
- Interactive carousels for browsing destinations

## Getting Started

### Prerequisites
- Node.js 20 or higher
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/ESNTurkiye/ESN-GO.git
cd ESN-GO
```

2. Navigate to the demo directory:
```bash
cd demo
```

3. Install dependencies:
```bash
npm install
```

4. Run the development server:
```bash
npm run dev
```

5. Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.

### Build for Production

```bash
npm run build
npm start
```
